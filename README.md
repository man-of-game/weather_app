# Python Weather App
A simple desktop application built with Python and Tkinter that fetches and displays the current weather for any city using the OpenWeatherMap API.

<h2>Features</h2>

* <b>Real-Time Data:</b> Fetches live weather data from the OpenWeatherMap API.
* <b>Simple UI:</b> A clean and user-friendly interface built with tkinter. 
* <b>Search Functionality:</b> Enter any city name to get its current weather. 
    * City Name
    * Current Temperature (in Celsius)
    * Weather Condition (e.g., "broken clouds", "clear sky")
* <b>Error Handling:</b> Includes pop-up alerts for common errors like "City not found" or "Network error."

<h2>Technologies Used</h2>

* <b>Python 3:</b> The core programming language.
* <b>Tkinter:</b> Python's standard library for creating the graphical user interface (GUI).
* <b>Requests:</b> A Python library used to send HTTP requests to the API.
* <b>OpenWeatherMap API:</b> The data source for all weather information.
